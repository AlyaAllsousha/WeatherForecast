#  WeatherApp (Jetpack Compose)

Минималистичное Android-приложение для просмотра погоды в любом городе мира. Написано на **Kotlin** с использованием **Jetpack Compose** и **REST API**.

![Kotlin](https://img.shields.io/badge/Kotlin-1.9.0-purple)
![Compose](https://img.shields.io/badge/Jetpack%20Compose-UI-green)
![API](https://img.shields.io/badge/REST%20API-Retrofit-blue)

##  Функционал
*   **Поиск по городу:** Пользователь вводит название города, приложение делает запрос к API.
*   **Текущая погода:** Отображение температуры, описания (ясно/облачно), влажности и скорости ветра.
*   **Динамические иконки:** Загрузка иконок погоды по URL (используя Coil).
*   **Обработка ошибок:** Уведомления, если город не найден или отсутствует интернет.

## Технологический стек
*   **Язык:** Kotlin
*   **UI:** Jetpack Compose (Material Design 3)
*   **Архитектура:** MVVM (Model-View-ViewModel)
*   **Сеть:** Retrofit2 + OkHttp + Gson Converter
*   **Асинхронность:** Kotlin Coroutines + Flow
*   **Загрузка изображений:** Coil
